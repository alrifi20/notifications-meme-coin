# notifications-meme-coin

A Web3 + AI mini project designed to simplify meme coin trading on 
[DexScreener](https://dexscreener.com/), providing accurate real-time 
monitoring 24/7 for meme coins on the Solana network.

This tool helps traders track price movements, trading volume, and 
market trends, making it easier to identify which meme coins are 
trending up or down at any given moment.

##Features
- Real-time meme coin price tracking on the Solana network via DexScreener
- 24/7 accurate market monitoring, including trading volume data
- Custom price alert setup — set a buy price and a target sell price 
  (e.g., buy at $0.003 and get notified when the price reaches $0.005)
- Automated notifications delivered directly through a **Telegram bot**
- Helps traders make faster, data-driven trading decisions
